# Fase 2 - Frontend Corporativo

Proyecto funcional en Next.js listo para GitHub y Vercel.

## Incluye
- Home profesional.
- Sección Prevención de Riesgos.
- Sección Masoterapia.
- Página Contacto.
- Navbar y Footer.
- Estilos globales corporativos.
- Preparado para Fase 3: reservas y calendario.

## Instalación local
```bash
npm install
npm run dev
```

## Vercel
- Framework Preset: Next.js
- Build Command: npm run build
- Output Directory: dejar vacío
