# Etapa 4 - Mejora Visual y Contenido Estratégico

Esta versión toma como base la Etapa 3 y mejora la presentación comercial de la APP-WEB.

## Cambios principales
- Home más visual y persuasiva.
- Nuevas secciones:
  - "Para empresas"
  - "Para personas"
  - "Proceso de trabajo"
  - "Beneficios"
  - "Servicios destacados"
  - "Bloques visuales tipo imagen"
  - "CTA comercial mejorado"
- Mejoras visuales:
  - Tarjetas con mayor jerarquía.
  - Bloques tipo imagen para reemplazar luego por fotografías reales.
  - Secciones alternadas con fondos corporativos.
  - Mejor presentación de precios.
- Mantiene:
  - Calendario profesional.
  - Supabase.
  - Reservas.
  - Panel administrador.
  - Recordatorios.
  - WhatsApp.
  - SEO.

## Variables necesarias en Vercel
- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `NEXT_PUBLIC_ADMIN_PIN`
- `NEXT_PUBLIC_WHATSAPP_NUMBER`
- `CONTACT_EMAIL`
- `CRON_SECRET`
