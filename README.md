# Etapa 1 - Landing Home Corporativa

Esta versión toma como base la Fase 7 y agrega la primera etapa de mejoras:

## Cambios principales
- Home extendida tipo landing page.
- Servicios de Prevención incorporados en la página principal.
- Servicios de Masoterapia incorporados en la página principal.
- Menú superior simplificado.
- Sección D.S. N°44.
- Sección CPHS.
- Sección de preguntas frecuentes.
- Sección de confianza y propuesta de valor.
- CTA final para agendar o contactar.
- Se eliminan del menú las páginas Prevención y Masoterapia como navegación principal.
