# Etapa 2 - Calendario Profesional

Esta versión toma como base la Etapa 1 y mejora la experiencia de agenda.

## Cambios principales
- Calendario visual por próximos 14 días.
- Selector de día tipo tarjeta.
- Visualización clara de horarios disponibles.
- Resumen de reserva antes de enviar.
- Mejor experiencia móvil.
- Panel admin mantiene gestión de disponibilidad.
- Las agendas siguen conectadas a Supabase.

## Se mantiene
- Landing principal de Etapa 1.
- Supabase.
- Panel administrador.
- WhatsApp.
- Contacto.
- Políticas.
- SEO.
- Sitemap.
- Robots.
