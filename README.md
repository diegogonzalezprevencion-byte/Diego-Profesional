# Fase 3 - Reservas Frontend

Proyecto Next.js funcional listo para GitHub y Vercel.

## Incluye
- Frontend corporativo de Fase 2.
- Agenda para Prevención de Riesgos.
- Agenda para Masoterapia.
- Selector de fecha, hora y servicio.
- Formulario de reserva.
- Confirmación visual en pantalla.
- Preparado para Fase 4: Supabase y persistencia real de reservas.

## Importante
Esta fase todavía no guarda datos en una base de datos. La reserva queda simulada en frontend.

## Uso local
```bash
npm install
npm run dev
```

## Vercel
- Framework Preset: Next.js
- Build Command: npm run build
- Output Directory: dejar vacío
